    This is the documentation overview page for the Cloudburo Curation Platform Product.

    This is currently work in progress.

    BTW: This documentation site is based itself on the Cloudburo Curation Platform and all content is mastered and maintained in Evernote’s One Workspace. It gives you an idea how you can use the Curation Platform to easily manager your documentation site as well.

    Overview

    Cloudburo’s Blogging Platform is a content curation solution which can be managed by the end user with minimal effort in the most efficient and intuitive manner, aimed at micro businesses, startups as well as private users. It’s a Amazon AWS Cloud-based service will seamlessly integrates into Evernote’s One Workspace ecosystem to provide you with the best user experience. Being based on Evernote it offers you a superior front-end in order to manage all your curation and blogging activities.

    Features

    The Curation Platform provides the following features

    • Bootstrap based Website Themes as provided by bootswatch.com
    • Syntax Highlighting Themes for rendering your Evernote code blocks in a note.
