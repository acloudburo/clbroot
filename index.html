    This is the documentation overview page for the Cloudburo Curation Platform Product.

    This is currently work in progress.

    BTW: This documentation site itself is based on the Cloudburo Curation Platform and all content is mastered and maintained in Evernote’s One Workspace. It gives you an idea how you can use the Curation Platform to easily manage your documentation site as well.

    Overview

    Cloudburo’s Blogging Platform is a content curation solution which can be managed by the end user with minimal effort in the most efficient and intuitive manner. Aimed at startups, micro businesses, small- and medium enterprises (SME) as well as private users, it will allow you to establish a content presence in the Internet for your audience.

    It’s a Amazon AWS Cloud-based service which seamlessly integrates into Evernote’s One Workspace ecosystem to provide you with the best user experience. Being integrated with Evernote it offers you a superior front-end in order to manage all your curation and blogging content with an easy to use tool and workflow.

    You will use the power of Evernote to manage and collect great sharing content, to compile it and then share it via the Cloudburo Curation Platform in a nicely, mobile-friendly designed blogging site. Concentrate in collecting and compiling content and leave the publishing to the Curation Platform, which ensures a great user experience on any kind of device.

    Features

    The Curation Platform provides the following features

    • Responsive, Mobile Optimized Content Website: The content website will follow the mobile first approach and be responsive. This ensures that your content is delivered to any device in an optimal layout.
    • Bootstrap based Website Themes: Your content will be published in a dedicated Website which is designed upon Bootstrap. It’s the most popular HTML, CSS and JS framework for developing responsive, mobile friendly sites. You will have the choice to select one of several predefined Cloudburo curation themes, which were developed by bootswatch.com.
    • Syntax Highlighting Themes: You have the possibility to define code blocks in your Evernote notes which will nicely converted into Syntax Highlighted content on your Website. A Pigment compatible syntax highlighter is chosen for this step (the de facto standard for syntax highlighting).
    • Image Handling: Pictures in Evernote notes will be taken over and converted for fast online access. Various possibilities exists how to control the positioning and layout of the image. Tutorial.
    • Video Integration: YouTube based videos can be easily embedded by adding some meta information in your Evernote note. Refer to the Tutorial , as well check the layout by opening a documentation page with a video integrated on this site (remember the whole documentation site is based on the Cloudburo Curation Platform).
    • W3C Atom 1.0 valid RSS Feeds: The Cloudburo Curation Platform managed content sites are providing a Atom 1.0 valid RSS feed of your content. This allows you to embed your content in any kind of RSS feed based third party applications.

    Some small tips which will improve your efficiency and quality when working with Evernote and the Curation Platform.

    • Use the Evernote Spell Checker as a content quality checker. High quality content with correct spelling is important and is influencing your ranking in search engines like google (SEO = Search Engine Optimization).
