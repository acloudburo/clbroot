Feature: Customizable Content Site Index Page

Posted on 22.May 2015 | Tags pubbot.siteconfig



By default the Article List page is shown as your index page of your content site. You can see a screenshot below how that looks like.
But you have the possibility to define a own created simple index page. This can be managed as any other site directly in Evernote.


Via the Memberzone Dashboard (Setup Tab) you can switch on a customized index page.



When choosing this option, the Cloudburo Curation Platform will generate a note with the tag „clb-index“.


You can define the content completely freely. For example this documentation site uses this feature. The Article List Page will be available via a dedicated menu entry „Articles"
